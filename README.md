# Netflix Movies and TV Shows — Data Analysis

## Project Overview
Exploratory data analysis of Netflix's content library using Python.
Dataset: Netflix Movies and TV Shows by Shivam Bansal (Kaggle)
Total titles analyzed: 8,794

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
1. Data Cleaning — handled 3,000+ missing values
2. Movies vs TV Shows — distribution and percentages
3. Content Growth Over Years — trend analysis
4. Top Countries — which countries produce the most content
5. Genre Analysis — top 15 genres on Netflix
6. Movie Duration Stats — mean, median, std, IQR, outliers
7. India Focus — deep dive into Indian content on Netflix

## Key Findings
- Netflix is 69.7% movies and 30.3% TV shows
- USA dominates with 31.94% of all content
- India is #2 with 11.05% — driven by Bollywood licensing
- Netflix peaked in 2019 with 2,016 titles added in one year
- Average Netflix movie is ~99 minutes long
- International Movies and Dramas are the top genres
- India's Netflix content peaked in 2018 with 330 titles in one year
